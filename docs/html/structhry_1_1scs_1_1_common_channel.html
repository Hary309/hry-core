◆ localScale

float hry::scs::CommonChannel::localScale

Scale applied to distance and time to compensate for the scale of the map (e.g. 1s of real time corresponds to local_scale seconds of simulated game time).

Games which use real 1:1 maps will not provide this channel.

◆ restStop

int32_t hry::scs::CommonChannel::restStop

Time until next rest stop.

When the fatique simulation is disabled, the behavior of this channel is implementation dependent. The game might provide the value which would apply if it was enabled or provide no value at all.

Represented in in-game minutes.
